Problem Statement : ReWear – Community Clothing Exchange

> A web-based platform that promotes sustainable fashion through community-driven clothing swaps and point-based redemption. Built during the **Odoo Hackathon '25**.
---

## 🌱 Overview

**ReWear** enables users to give their unused clothing a second life. Instead of throwing away wearable clothes, users can:
- 🌀 Swap directly with others
- 🪙 Redeem clothing using points
- 🔁 Encourage eco-friendly fashion

This platform is designed with simplicity, accessibility, and sustainability in mind.

---

## 🧩 Features

### 👤 User Features
- Sign up / Login with email or Google
- Create a profile with points tracking
- List items with images, size, condition, and tags
- Browse/search items using filters
- Send swap requests or redeem items with points
- View dashboard: uploaded items, ongoing/completed swaps

### 🛠️ Admin Features (Optional)
- Moderate item listings (approve/reject)
- Remove flagged or inappropriate items
- Lightweight dashboard for reporting
